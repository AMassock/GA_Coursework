# OOP Cards

Use your new understanding of object-oriented programming in JavaScript to build
the foundation for a card game!

## Prerequisites

- Objects and context
- Object-oriented programming and defining classes

## Instructions

1.  Fork and clone this repository.
1.  Change into the new directory.
1.  Fulfill the listed requirements.

Starter code is available in [`lib/challenge.js`](lib/challenge.js). A pull
request is not required, but it is necessary if you want a code review.

## Requirements

Read through the prompt in [`lib/challenge.js`](lib/challenge.js). You'll be
building out two classes in JavaScript:

* `Card`
* `Deck`

Each class will have attributes and methods, as described in the prompt.
