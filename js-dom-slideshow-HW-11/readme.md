# JS DOM Slideshow

Use the JavaScript DOM techniques that we've covered so far to build
a slideshow.

## Prerequisites

- JavaScript
- HTML and CSS
- The DOM

## Instructions

1. Fork and clone this repository.
1. Change into the new directory.
1. Fulfill the listed requirements.

Starter code is available in [`lib/`](lib/). You are required to turn in your
submission by making a pull request on the original repository.

## Requirements

Work through the instructions provided in `lib/script.js` to build out a slider
using JavaScript.
