# Arrays Practice

Use the challenges in this repository to get some practice with working with
Arrays in JavaScript!

## Instructions

1.  Fork and clone this repository.
1.  Change into the new directory.
1.  Fulfill the listed requirements.

Starter code is available in [`lib/challenge.js`](lib/challenge.js). Turn in
your work by making a pull request on this repository.

## Requirements

Work through the prompts in [`lib/challenge.js`](lib/challenge.js).
